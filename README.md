# Langkah Menjalankan Program App Movie List
Langkah pertama install terlebih dahulu npm :
```
npm init
```

```
npm install
```

Setelah menginstall npm , lamgkah selanjutnya adalah menjalankan program app, dengan cara:
```
npm run start
```

# Validasi Hasil
